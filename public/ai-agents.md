# AI-agents — Code Lieshout

> AI-agents zijn autonome systemen die zelfstandig complexe taken uitvoeren. Code Lieshout implementeert ze voor het Nederlandse MKB.

## Wat is een AI-agent

Een AI Agent is een autonoom systeem dat zelfstandig taken kan uitvoeren, beslissingen kan nemen en kan leren van interacties. Anders dan een chatbot kan een agent meerdere stappen plannen en uitvoeren zonder directe tussenkomst van een mens.

## Chatbot vs AI-agent — het verschil

- **Chatbot**: reageert op directe vragen, voert enkelvoudige taken uit, geen eigen initiatief
- **AI-agent**: handelt complexe meerstaps-taken autonoom af, gebruikt tools, neemt beslissingen op basis van context en doelen

## Wat kan een AI-agent voor jouw bedrijf doen

- E-mails beheren, categoriseren en beantwoorden
- Agenda's plannen en meetings voorbereiden
- Data analyseren en rapporten genereren
- Code schrijven, reviewen en debuggen
- Onderzoek doen op het web en samenvatten
- Administratieve workflows automatiseren
- Klantvragen routeren en afhandelen

## Is het veilig

Ja, mits goed geconfigureerd. AI-agents kunnen worden ingesteld met strikte permissies, audit logs, en menselijke goedkeuring voor kritieke acties. Privacy en beveiliging zijn ingebouwd vanaf ontwerp.

## Hoe start je

1. **Strategie-gesprek** — samen kijken welke processen kandidaat zijn voor automatisering
2. **Proof of concept** — klein, zichtbaar resultaat in 1-2 weken
3. **Productie-implementatie** — opschalen met monitoring en veiligheidsmaatregelen
4. **Onderhoud** — doorlopende verbetering en aanpassing aan nieuwe AI-ontwikkelingen

## Frameworks en standaarden

Code Lieshout werkt met moderne AI-protocollen:

- **MCP (Model Context Protocol)** — standaard voor tool-exposure aan AI-agents
- **WebMCP** — browser-native agent integration via `navigator.modelContext`
- **Agent Skills Discovery** — standaard voor het publiceren van uitvoerbare skills
- **A2A (Agent-to-Agent)** — agent-communicatie standaarden
