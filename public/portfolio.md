# Portfolio — Code Lieshout

> Recente projecten en werkervaring van Pim Lieshout.

## Wat je in dit portfolio vindt

- Recente websites en web-applicaties gebouwd met Next.js en React
- AI-agent implementaties en automatiserings-projecten
- Case studies met resultaten
- Werkervaring en achtergrond

## Werkwijze

Elk project start met begrip van de business. Geen technologie om de technologie, maar pragmatische keuzes die aansluiten op wat een klant nodig heeft. Iteratief bouwen met korte feedback-cycli.

## Technologie-stack

- **Frontend**: Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS
- **3D en animatie**: Three.js, GSAP, Framer Motion / Motion
- **Backend**: Next.js Route Handlers, Supabase (Postgres), Node.js
- **AI**: Claude API, OpenAI API, MCP, custom agent-orchestratie
- **Deploy**: Vercel, edge-runtime, serverless
