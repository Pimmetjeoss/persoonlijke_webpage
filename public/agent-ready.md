# Agent-Ready Scan — Code Lieshout

> Hoe vindbaar en bruikbaar is jouw website voor AI-agents? Gratis scan in 10 seconden tegen 19 actuele standaarden — geen account nodig.

## Waarom agent-readiness

AI-agents zoals ChatGPT, Claude en Perplexity halen steeds vaker informatie op namens hun gebruikers en doen aanbevelingen. Een site die voor agents goed leesbaar is, wordt vaker correct begrepen, geciteerd en aanbevolen. Agent-readiness is de nieuwe laag bovenop klassieke SEO.

## Wat de scan controleert

De gratis scan toetst een website tegen 19 actuele agent-readiness standaarden, waaronder:

- **llms.txt** — een index voor AI-agents
- **Markdown content negotiation** — pagina's opvraagbaar als `text/markdown`
- **MCP (Model Context Protocol)** en **WebMCP** — tool-exposure aan agents
- **Agent Skills Discovery** — publiceren van uitvoerbare skills
- **OAuth discovery** voor agents
- **Structured data en schema.org** — machine-leesbare betekenis
- **Crawler-toegang** voor GPTBot, ClaudeBot, PerplexityBot en andere AI-crawlers
- **Sitemaps en robots-configuratie**

Je krijgt direct een score en concrete verbeterpunten.

## Hoe het werkt

1. Voer een website-URL in op [code-lieshout.nl/agent-ready](https://code-lieshout.nl/agent-ready)
2. Binnen ongeveer 10 seconden verschijnt een score met bevindingen
3. Bekijk per standaard wat goed is en wat beter kan
4. Optioneel: laat Code Lieshout de verbeteringen implementeren
