# Code Lieshout — Pim Lieshout

> Webontwikkelaar en AI-specialist uit Nederland. Bouwt agent-ready websites, slimme web-applicaties en AI-automatiseringen voor het MKB.

## Wat Code Lieshout doet

- **Websites op maat** — moderne Next.js/React sites met aandacht voor SEO, performance en agent-readiness
- **AI-agents en automatisering** — van chatbots tot autonome agent-workflows die echt werk uit handen nemen
- **Web-applicaties** — maatwerk software die past bij jouw bedrijfsproces
- **Agent-readiness audits** — scan en optimalisatie van je site voor ChatGPT, Claude, Perplexity en andere AI-agents

## Waarom Code Lieshout

Pim Lieshout combineert traditionele webontwikkeling met de nieuwste AI-standaarden: MCP (Model Context Protocol), WebMCP, Agent Skills Discovery, OAuth discovery voor agents, en Markdown for Agents. Jouw website is niet alleen klaar voor bezoekers, maar ook voor de AI-agents die steeds vaker aanbevelingen doen namens hun gebruikers.
