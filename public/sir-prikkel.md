# Sir Prikkel — Code Lieshout

> Sir Prikkel is de persoonlijke AI-assistent ("digitale schildknaap") van Code Lieshout — een levend voorbeeld van een maatwerk AI-agent.

## Wie is Sir Prikkel

Sir Prikkel is een op maat gebouwde AI-agent die zelfstandig taken uitvoert, onthoudt wat belangrijk is en samenwerkt met de tools die je dagelijks gebruikt. Hij laat concreet zien wat Code Lieshout voor jouw organisatie kan bouwen.

## Wat Sir Prikkel kan

Een greep uit zijn skills:

- Taken en workflows zelfstandig uitvoeren
- Documenten en informatie verwerken en samenvatten
- Onderzoek doen en context verzamelen
- Schakelen tussen meerdere tools en services

## Integraties

Sir Prikkel kan verbinden met een breed scala aan tools en services, en daarnaast met custom integraties en API's op maat.

## Onder de motorkap

- **Geheugen** — meerdere vormen van geheugen, inclusief semantic search om relevante context terug te vinden
- **Veiligheid is hard gecodeerd in kernregels:**
  - Geen data-exfiltratie — privézaken blijven privé
  - Geen destructieve acties zonder toestemming
  - Geen externe communicatie zonder akkoord
  - Bij twijfel: altijd vragen
